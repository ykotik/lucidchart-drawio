# gcp-data-platform

Build a GCP data platform streaming & batch pipeline: Sources (Cloud SQL, Firestore, Pub/Sub Ingestion, Cloud Storage Raw) -> Processing (Dataflow Batch, Dataflow Stream, Dataproc Spark) -> Storage (BigQuery, Cloud Storage Processed, Bigtable) -> Consumers (Looker Dashboards, Vertex AI, Data Catalog, Cloud Functions Alerts). Use GCP icons.
