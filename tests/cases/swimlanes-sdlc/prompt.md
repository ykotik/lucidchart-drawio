# swimlanes-sdlc

Create an SDLC toolchain swimlanes diagram with 5 phases (Plan, Develop, Build, Test, Deploy) containing: Jira, Confluence, VS Code, GitHub, SonarQube, Jenkins, Docker, Artifactory, Selenium, k6, Snyk, Trivy, ArgoCD, Kubernetes, Datadog, PagerDuty, and Terraform.

Use the swimlanes pattern.
