# aws-3tier-webapp

Create an AWS 3-tier web app architecture. Inside a VPC container: Public Subnet (ALB, NAT Gateway, Bastion), Private App Subnet (ECS Fargate Cluster, ElastiCache Redis), Private Data Subnet (RDS Aurora Primary, RDS Aurora Replica, DynamoDB). External: Route 53, CloudFront, S3 Assets, Cognito, CloudWatch, and SNS. Use AWS icons.

Use the scope-columns pattern.
