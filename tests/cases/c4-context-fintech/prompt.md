# c4-context-fintech

Build a C4 Context diagram for the fintech platform 'PayFlow Platform'. It includes 3 user personas (Customer, Merchant, Admin), the core PayFlow system, and 8 external systems (Payment Processor, KYC Provider, Fraud Detection, Banking API, Email Service, SMS Gateway, Analytics Platform, Regulatory Reporting). Define relationships for all interfaces.

Use the c4-context pattern.
