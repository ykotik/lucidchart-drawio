# bpmn-order-process

Create a BPMN order fulfillment process diagram. Pool: 'Order Management' with 3 lanes: Sales, Warehouse, Shipping. Process includes: Start event, Validate Order, check-credit gateway, Pick Items, Pack Items, parallel-packaging gateway, Generate Label, ship-method gateway, Ship Standard, Ship Express, Send Invoice, Update Status, Notify Customer, SLA breach event, and End event.

Use the bpmn-process pattern.
